# Contributing to AI Knowledge Commons

Thank you for considering a contribution! This project is a community-owned,
open educational resource, and every improvement helps more people learn.

## Ways to Contribute

- **Fix or improve an explainer** in `content/` (clarify wording, correct a
  factual error, add a citation).
- **Add a new topic** as a new Markdown file in `content/`, numbered
  sequentially (e.g. `04-...md`).
- **Improve tooling** in `tools/`.
- **Extend the resource list** in `resources/`.

## Before You Open a Pull Request

1. Fork the repository and create a topic branch.
2. Keep changes focused and self-contained.
3. For factual claims, add a citation link where possible.
4. Ensure any new code runs with `python tools/explain_concept.py`.

## Content Guidelines

- Write for a general audience; avoid unexplained jargon.
- Stay neutral and accurate; this is an educational project, not advocacy.
- Release your text under CC BY 4.0 by adding the license note at the top of
  the file.
