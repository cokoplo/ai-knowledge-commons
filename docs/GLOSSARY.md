# Glossary

Key terms from the AI Knowledge Commons materials.

- **Artificial Intelligence (AI)** — The field of building systems that perform tasks we usually associate with human intelligence, such as understanding language or recognizing patterns.
- **Machine Learning** — A subset of AI where systems learn patterns from data instead of being given explicit step-by-step rules.
- **Model** — A trained system (e.g., a neural network) that turns inputs into outputs; in LLMs the learned numbers are called weights.
- **Training** — The process of feeding data to a model so it adjusts its weights to make better predictions.
- **Inference** — Using a trained model to produce output on new inputs; the "asking the model a question" phase.
- **Large Language Model (LLM)** — A model trained on vast text to predict the next token; the engine behind most modern chat assistants.
- **Token** — A chunk of text (roughly a word or part of a word) that models read and generate; billing and limits are usually counted in tokens.
- **Weights** — The numerical parameters a model learns during training; "open weights" means these are published for anyone to run.
- **Fine-tuning** — Further training a pre-trained model on a narrower dataset to adapt it to a specific task.
- **Hallucination** — When a model confidently produces false or fabricated information; a known limitation, not a feature.
- **Prompt** — The instruction or input you give a model; prompt design strongly affects output quality.
- **Context window** — The maximum amount of text (tokens) a model can consider at once, including your prompt and its reply.
- **Bias** — Systematic skew in model output that reflects patterns in its training data; can reinforce stereotypes.
- **Open source** — Software whose source code is published under a license that permits free use, study, and modification.
- **Open weights** — A model whose trained parameters are publicly released, so anyone can run or inspect it.
- **C2PA** — An open standard for attaching provenance metadata that can mark whether content was AI-generated.
- **RAG (Retrieval-Augmented Generation)** — A technique where a model fetches relevant documents before answering, to ground responses in real sources.
- **Guardrails** — Constraints or filters that steer a model's behavior toward safe and intended use.
- **AI literacy** — The ability to understand, evaluate, and use AI systems responsibly as a member of the public.
