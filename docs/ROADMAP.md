# Roadmap

*AI Knowledge Commons — public roadmap (living document)*

This roadmap is intentionally public so that contributors and fiscal sponsors can
see where the project is headed. It is non-binding and updated as we learn.

## What we have shipped

- ✅ Core explainer series: what AI is, how LLMs work, AI literacy & ethics.
- ✅ Practical guides: spotting AI-generated content, responsible use, open models.
- ✅ Open-source tooling: `explain_concept.py` and `build_glossary.py` (no dependencies).
- ✅ Automated checks (CI) that run our tools on every push.

## Near term

- [ ] A short, illustrated "AI literacy 101" one-pager for classrooms.
- [ ] Translations of the core explainers (starting with Chinese and Spanish).
- [ ] A community-contributed glossary review process.

## Longer term

- [ ] Interactive examples that run in the browser without installs.
- [ ] A contributor onboarding path for first-time open-source participants.
- [ ] Periodic "state of AI literacy" notes summarizing new public research.

## How to influence this

Open an issue or discussion. Because all materials are openly licensed, anyone can
propose additions — that is the point of the project.

## Principles that guide priorities

1. **Public good first.** We optimize for free, accessible learning, not engagement metrics.
2. **Open by default.** Code, text, and tooling stay openly licensed and runnable.
3. **Accuracy over hype.** We would rather say "we don't know" than overclaim.
